## Secure Api Key 
Secure api key with server side, get request from Android side with public key. Server side is using Ktor and Kobweb frameworks.

## Dependencies App
- Jetpack Compose
- Retrofit
- Dragger Injection
- KotlinX Serialization
- Shared Preferences

## Dependencies for Server
- Ktor
- Kobweb
